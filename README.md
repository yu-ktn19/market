# market
# 環境構築
## Dockerビルド
### git clone github.com/yu-ktn19/market
### docker-compose up -d --build

## Laravel環境構築
### docker-compose exec php bash
### composer install
### cp.env.example.evn
### php artisan key:generate
### php artisan migrate
### php artisan db:seed
